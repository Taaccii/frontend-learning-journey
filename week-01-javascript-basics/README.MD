- `const` per costanti
- `var` evitato (vecchio standard)

### Funzioni
```javascript
// Function declaration
function sum(a, b) {
  return a + b;
}

// Arrow function
const sum = (a, b) => a + b;
```

### Arrays
```javascript
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(n => n * 2);
const even = numbers.filter(n => n % 2 === 0);
```
